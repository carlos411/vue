# 4.3 template 中的 ref

## 透過 ref 來取得 DOM 元素

建立 **`practice/template_ref.html`** 檔案，內容如下：

```html
<!DOCTYPE html>
<html lang="zh-Hant">
  <head>
    <meta charset="utf-8">
    <title></title>
  </head>
  <body>

    <div id="app">
      <input type="text" id="my_id" ref="username">
    </div>

    <script src="./vendors/vue/vue.global.js"></script>
    <script>
      const app = Vue.createApp({
        mounted(){
          this.$refs.username.focus();
          console.log(this.$refs.username.id);
        }
      });

      app.mount("#app");
    </script>
  </body>
</html>
```
